# Примеры файлов данных

Эта директория содержит примеры входных файлов для тестирования и обучения.

## Доступные файлы

### sample_coordinates.txt
Пример файла в формате TXT с заголовками и комментариями.
- 25 точек
- Формат: X Y Z с заголовком
- Комментарии с информацией о съемке

### sample_coordinates.xyz
Тот же набор данных в формате XYZ (без заголовков).
- 25 точек
- Формат: X Y Z (только координаты)

## Использование

### В боте
1. Запустите бот командой `/new`
2. Отправьте один из примеров файлов
3. Следуйте инструкциям бота для обработки

### Локально для тестирования
```python
import numpy as np

# Загрузка данных
data = np.loadtxt('examples/sample_coordinates.txt', skiprows=5)
print(f"Загружено точек: {len(data)}")
print(f"Диапазон X: [{data[:, 0].min()}, {data[:, 0].max()}]")
print(f"Диапазон Y: [{data[:, 1].min()}, {data[:, 1].max()}]")
print(f"Диапазон Z: [{data[:, 2].min()}, {data[:, 2].max()}]")
```

## Создание собственных тестовых данных

Вы можете создать свои тестовые файлы, следуя формату примеров.

### Генерация синтетических данных

```python
import numpy as np

# Генерация регулярной сетки с шумом
x = np.linspace(1000, 1020, 10)
y = np.linspace(2000, 2020, 10)
X, Y = np.meshgrid(x, y)

# Добавление "холма" и шума
Z = 150 + 0.5 * np.sin(X/10) + 0.5 * np.cos(Y/10) + np.random.normal(0, 0.05, X.shape)

# Сохранение
with open('custom_data.txt', 'w') as f:
    f.write('# Custom test data\n')
    f.write('X Y Z\n')
    for xi, yi, zi in zip(X.flatten(), Y.flatten(), Z.flatten()):
        f.write(f'{xi:.3f} {yi:.3f} {zi:.3f}\n')
```
