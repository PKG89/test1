# Скриншоты для документации

Эта директория предназначена для хранения скриншотов, используемых в документации.

## Необходимые скриншоты

### AutoCAD

1. `autocad-new-template.png` - Создание нового шаблона (File → New)
2. `autocad-units.png` - Диалог настройки единиц измерения
3. `autocad-layers.png` - Layer Properties Manager
4. `autocad-text-style.png` - Text Style Manager
5. `autocad-block.png` - Block Definition диалог
6. `autocad-border.png` - Пример рамки и штампа ГОСТ

### Примеры работы бота

7. `bot-welcome.png` - Приветственное сообщение бота
8. `bot-processing.png` - Процесс обработки данных
9. `bot-result.png` - Готовый результат

### Примеры выходных файлов

10. `output-preview.png` - Превью сгенерированного чертежа
11. `tin-example.png` - Пример триангуляционной сети

## Формат

- Формат: PNG
- Разрешение: минимум 1280x720
- Качество: высокое, без артефактов сжатия
- Язык интерфейса на скриншотах: предпочтительно русский

## Добавление скриншотов

Разместите файлы в этой директории с соответствующими именами, указанными выше.
